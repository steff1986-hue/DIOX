# Setup Strumenti Internet MCP (Italiano)

## Stack iniziale consigliato
1. Brave Search MCP
   - ricerca web/notizie/locale come strumento principale
2. Fetch MCP
   - lettura e sintesi URL
3. Time MCP
   - data/ora corrente e fusi orari

## Regole operative
- API key sempre fuori da Git.
- Avvia solo questi 3 strumenti nella prima fase.
- Aggiungi strumenti di ricerca profonda solo se necessario.
- Non assumere formati config non verificati: usa esempi e passaggi UI manuali.

## Sequenza raccomandata
1. collega MCP endpoint XiaoZhi al bridge
2. aggiungi Brave Search MCP
3. aggiungi Fetch MCP
4. aggiungi Time MCP
5. testa risposte con richieste aggiornate nel tempo
6. conferma citazioni/fonti quando supportate a runtime
