# Setup Live XiaoZhi Console (Italiano)

1. Apri XiaoZhi Console e seleziona agente da configurare.
2. Imposta nome assistente: `DIOX`.
3. Imposta lingua dialogo: `Italiano`.
4. Se disponibile, seleziona voce maschile italiana.
5. Copia/incolla prompt da `diox/prompts/diox-role-introduction.md`.
6. Mantieni memoria `short-term` durante fase test.
7. Dopo ogni cambio modello importante, pulisci memoria.
8. Esegui primo test senza MCP (identita, stile, risposte brevi).
9. Solo dopo test base, collega MCP endpoint.
10. Salva configurazione e riavvia sessione/dispositivo se richiesto.

## Nota pratica
- Se etichette UI cambiano nel tempo, cerca campi equivalenti: Name, Language, Voice, Prompt, Memory, MCP Endpoint.
