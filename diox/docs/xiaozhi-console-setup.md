# XiaoZhi Console Setup For DIOX

1. Open XiaoZhi console and select/create assistant.
2. Set assistant name to `DIOX`.
3. Set dialogue language to Italian.
4. Select Italian male voice if available.
5. Replace default role prompt with content from `diox/prompts/diox-role-introduction.md`.
6. Keep short-term memory during early tests.
7. Keep answers short for voice use.
8. Benchmark available models before choosing default.
9. Clear memory after major model changes.
10. Do not enable advanced memory until prompt and model are stable.
