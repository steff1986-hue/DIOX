# Internet Access Architecture

Model knowledge alone is not reliable for current information.
Internet access must run through MCP tools.

## Preferred Minimal Stack
1. Brave Search MCP for web, news, local, image/video, and general search.
2. Fetch MCP for reading/summarizing pages.
3. Time MCP for current date/time/time zones.
4. Optional Tavily or Firecrawl for deeper research/extraction.

## Principle
- Avoid too many tools at start.
- Tool overload increases latency and complexity.
- Start with search + fetch + time.
