# Personal Assistant Safety

- No secrets in repository.
- No raw tokens in docs.
- No real endpoint tokens.
- Internet results can be wrong; verify critical facts.
- Prefer source-grounded answers.
- No autonomous purchases.
- No payment actions.
- No destructive actions.
- No unrestricted browsing automation without explicit approval.
- No private-account access without separate security review.
- Memory must avoid sensitive information.
